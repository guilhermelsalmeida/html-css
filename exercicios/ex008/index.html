Principais formatações.


Negrito ou destaque.

Nesta frase há um termo em negrito usando a tag B, que é a forma não semântica.

Nesta frase há um termo em destaque usando a tag strong, que é a maneira semântica.


Itálico ou ênfase.

Nesta frase, há um termo em itálico usando a tag i (não semântica).

Nesta frase há um termo em ênfase utilizando a tag EM (semântica).


Texto marcado.

É possível criar, ainda, um texto marcado usando a tag MARK.

E neste outro parágrafo, há outro texto marcado no final.


Texto grande e pequeno.

Estão sendo criados um texto grande e um texto pequeno neste parágrafo.

Texto deletado.

É possível marcar um texto como excluído para indicar que deve ser lido, mas não considerado.


Texto inserido.

É possível marcar um texto como inserido a fim de dar ênfase e indicar que foi adicionado depois.

Existe também o sublinhado através da tag U, que é não semântica.


Texto sobrescrito.

Serve para inserir coisas como: ax2 + bx + c = 0.


Texto subscrito.

Serve para inserir coisas como: H2O.
